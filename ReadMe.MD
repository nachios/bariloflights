## Variables de Entorno

Las variables deben ser configuradas en el archivo `.env`. En este proyecto nos encontramos con credenciales de la RapidAPI, la CurrencyAPI, Airflow, y la base de datos.

- `RAPIDAPI_KEY`
- `RAPIDAPI_HOST`
- `RAPIDAPI_BASE_URL`
- `REDSHIFT_USER`
- `REDSHIFT_PASSWORD`
- `REDSHIFT_HOST`
- `REDSHIFT_PORT`
- `REDSHIFT_DB`
- `REDSHIFT_SCHEMA`
- `CURRENCYAPI_TOKEN`
- `AIRFLOW_UID`
