## Variables de Entorno

Las variables deben ser configuradas en el archivo `config.py`. En este proyecto nos encontramos con credenciales de la RapidAPI, la CurrencyAPI, y la base de datos.

- `RAPIDAPI_KEY`
- `RAPIDAPI_HOST`
- `RAPIDAPI_BASE_URL`
- `POSTGRES_USER`
- `POSTGRES_PASSWORD`
- `POSTGRES_HOST`
- `POSTGRES_PORT`
- `POSTGRES_DB`
- `POSTGRES_SCHEMA`
- `CURRENCYAPI_TOKEN`
